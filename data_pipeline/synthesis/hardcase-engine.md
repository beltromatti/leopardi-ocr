# Hard-Case Engine

Date locked: 2026-04-08

Leopardi should manufacture pressure where current frontier parsers still break.

## Core Hard-Case Families

### 1. Rotation And Mixed Orientation

Build from:

- exact pages
- tables
- formulas
- receipts and forms

Need:

- full-page arbitrary rotation
- local block rotation
- mixed orientations inside the same page

### 2. Formula-Dense Scientific Pages

Build from:

- arXiv
- PMC
- formula specialist pools

Need:

- inline plus display math mixture
- math near tables and captions
- small-font equations

### 3. Complex Tables

Build from:

- PubTables-1M
- SciTSR
- optional verified financial tables
- exact scientific pages containing tables

Need:

- merged cells
- multi-row headers
- dense numeric columns
- table plus surrounding paragraph interaction

### 4. Handwriting Mixed With Structured Documents

Build from:

- IAM, Bentham, READ 2016
- exact pages from arXiv and PMC

Need:

- margin notes
- underlines
- correction marks
- handwritten overlays on printed pages

### 5. Scan And Photo Degradation

Build from:

- exact pages
- forms and receipts

Need:

- blur
- low contrast
- compression artifacts
- shadow
- perspective distortion
- bleed-through style noise

### 6. Graphics-Heavy Pages

Build from:

- ChartQA
- PlotQA
- exact papers with figure-rich pages

Need:

- chart text near paragraphs
- captions tied to graphics
- figure plus equation coexistence

## Synthesis Philosophy

The pipeline should not overspend on fantasy documents.
It should overspend on realistic combinations that public benchmarks underrepresent.
