# Research Hub

This directory is the competitive-intelligence and benchmarking knowledge base for Leopardi.

## Main Documents

- `competitor-landscape-2026-04.md`: current market and research map of OCR/document parsing competitors.
- `leaderboard-2026-04.md`: first-pass ranking and benchmark summaries.
- `unified-metrics.md`: harmonized evaluation framework across fragmented benchmarks.
- `model-dossiers.md`: architecture, training, strengths, and risks for the main competitors.
- `sources.md`: source registry with evidence quality notes.
- `research-log-2026-04-07.md`: working notes from the first deep research pass.
